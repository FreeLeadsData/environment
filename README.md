## Installation Scripts

```bash

bash install.ubuntu.18-04.sh
```

**Ubunto 20.04:**

```bash
wget https://raw.githubusercontent.com/FreeLeadsData/environment/main/sh/install.ubuntu.20-04.sh

bash install.ubuntu.20-04.sh
```
